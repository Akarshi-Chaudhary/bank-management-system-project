
# 🏦 Bank Management System

This is a desktop-based **Bank Management System** I built using Java and Swing to simulate real-world banking operations like account handling, transactions, and user authentication.



## 🚀 Features

* 🔐 Secure Login System
* 💰 Deposit and Withdrawal functionality
* 📊 Balance Inquiry
* 🧾 Mini Statement generation
* 🖥️ User-friendly GUI using Java Swing



## 🛠️ Tech Stack

* Java (Core + OOP Concepts)
* Swing (for GUI)
* MySQL (for database)
* JDBC (for database connectivity)



## 📂 Project Structure
